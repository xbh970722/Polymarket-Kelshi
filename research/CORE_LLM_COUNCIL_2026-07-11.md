## 决定性证据 (五席交叉证实)
1. **一个机制连开三枪** (非孤例): 按事前 |q_LLM − 市场| 切: ≤12pt 6笔 6/6 全胜 +$0.93; >12pt 3笔 0/3 全亏 −$4.87 (GPT-5.6 −$4.53 + BEH −$0.34, 同"digest 缺决定性事实")。
2. **"救回内核"是幻觉**: 除 GPT 的 +$0.59, bootstrap CI [−$0.74,+$2.12] 跨零; PLATNER 一笔占 68%, 去掉=−$0.04。
3. **代码层坐实**: Kelly sizing 让下注额随 |LLM−市场| 单调递增(越可能错下越重); 唯一"分歧门"测 LLM 互相是否一致(engine.py:33), 非 vs 市场; `_half_anchor_ok`+`market_price_seen=false` 主动抹掉"我是离群值"信号; 验证层不被下单代码读取=非 veto。
4. **学术佐证**: KalshiBench 300 题 LLM Brier skill 多为负; 唯一正收益机制是小注 favorite-longshot("错时亏得少"), 与 CORE"越自信越重仓"相反。
5. **活雷**: MEDNOM (q0.19 vs 市0.52, 32pt, $2.45) 仍 open, 同型于 GPT。

## Fable 总裁决
- **STANCE=A (结构性负 EV)**, 5/5 收敛。LLM 信息集 ⊆ 市场信息集, 流动时事盘上"大分歧"≈自己落后=逆选择; 加分歧门只截尾不造 alpha。
- **动作①(自主降险)**: CORE 直赌盘暂停到 07-23(CLAUDE.md #10)。无 config 开关→靠纪律条款。在场 4 笔按既有 swing/止损跑, 点名 MEDNOM。
- **动作②(记提案, 07-23)**: 复活须先影子跑通: 分歧>15-20pt 跳过 + 快讯题(发布/宣布/发帖类)排除 + 验证层入场前 veto 权 + 规则精读白名单 + 单事件硬帽 5 张。
- **goal 级 pivot**: LLM 直接对盘口赌"我比市场懂未来" 不值得投钱。LLM 价值 = 结算规则律师(避 MAXMON 陷阱)/入场前否决错误前提/给量化盘出特征。**当参谋, 不当下注人。**
- 重开/加钱 = 用户专属。
